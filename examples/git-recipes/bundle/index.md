# Git Recipes

A small, practical knowledge bundle of the git operations developers reach
for under pressure: rewriting history before a push, recovering work that
seems lost, and juggling more than one line of work at once. Each file is a
single focused recipe — when to use it, the exact command, and the gotcha
that bites people.

Six recipes across three folders. Cross-links use OKF v0.1 §5.1 absolute
paths so the graph survives any mount point.

## Folder map

- [/rewriting/](rewriting/index.md) — reshape commits before they are shared
- [/recovery/](recovery/index.md) — get back work that looks gone
- [/workspace/](workspace/index.md) — hold or split work in progress

See [log.md](log.md) for the change history.
